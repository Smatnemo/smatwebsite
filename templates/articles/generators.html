Python Generators

Generator functions are special kind of functions that return lazy iterators. Unlike lists, lazy iterators do not store their contents in memory. Assuming you are familiar with what iterators are, let us proceed. Let us proceed to find out what they are like in action. Let's get to it.

Example 1: Reading Large Files

Working with large data streams or large files is a common use case. In our case, we will use a csv(comma-separated-values) file. A csv file separates data into columns using commas. Let's assume we have a function csv_reader that returns an iterator that we then pass to csv_gen. 

            csv_list = csv_reader("csv_file.txt")

            row_count = 0

            for row in csv_list:
                row_count += 1

            print(f"Row count is {row_count}")

Remember to create a csv_file.txt file in the current working directory. Add 3 or more words to each line and separate the words by commas. Each word represents a column. Do the same for as many lines or rows as you pleas. The function is defined below. 

            def csv_reader(filename):
                
                file = open(filename)
                result = file.read().split("\n")
                return result

The function opens a file "csv_file.txt" and uses read() function to read a file into a string. The split() function divides the string into items and adds them to a list. In my case, my csv file has five lines of text. The code snippet returned 5.
