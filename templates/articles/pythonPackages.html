Python Packages

Python packages are directories with the __init__.py file in their root directory. When you import a package, the __init__.py file runs first to initialize your package. In the package __init__.py file you can import modules or entities in the modules relatively. That way all the classes, functions and other entities in the modules you have within your package. Let's do all of these things in an example.

Example of Package imports

    Let's create a package with the following structure:
  1. Create a directory called package in your current working directory
  2. Change directory into your new directory
  3. Create a file named __init__.py
  4. Create two files named module1.py and module2.py

If you have done this as described above, you should have a package with such a structue:

package/
    __init__.py
    module1.py
    module2.py

In linux it, you can create it using the following:

tertese@systemname$~mkdir package
tertese@systemname$~cd package
tertese@systemname$~/package/ $touch __init__.py
tertese@systemname$~/package/ $touch module1.py
tertese@systemname$~/package/ $touch module2.py
tertese@systemname$~/package/ $cd ..
tertese@systemname$~/

If you have followed the instructions above, then your current directory should have the package directory listed. Use ls command to find out. Now create a python script or type python3 in your terminal and do the following test: 

tertese@systemname~$ python3
>>> import package
>>> import package.module1

If you have reached this far well done. However, we cannot do anything else. Not fun is it? Now use Ctrl + D to exit the python CLI.

Now let us add some useful python code in our package. Our package will be a math package. Use any code editor of your choice to edit module1.py.


import <package_name>

This syntax can be used to import a package
