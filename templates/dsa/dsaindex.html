Data Structures and Algorithms

Learning Data Structures and Algorithms is not only cool but it also helps to solve a lot of custom problems. Typically every problem you encounter has almost certainly been encountered by someone else before you. And they have-thank God- painstakingly spent a lot of time trying to solve it. As a result they came up with a template for such a problem and similar variations of the same problem. This will no doubt help you solve the problem, not just efficiently but in a robustly.

Now, the default programming language for this will be Python programming language. Yes, that's because it looks like an English description of the algorithms and hopefully, it will be easy to understand. Let's Get Started
